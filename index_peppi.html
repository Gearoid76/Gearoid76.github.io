Project Pia

Overview

In this project we the team of three, where responsible for creating an app that students could use to organise their class schedules, thought we would but it into a monthly and weekly event type calender, so that students could organise their life.

Problem - Organising courses into a timetable is complicated and time-consuming. Solution - Pia - a timetable app (for UAS).
